# Weather Dashboard.

---

## Purpose
A web application to search for cities around the world and get their current weather information and a 5 day forecast. Searched cities saved to page and persist on page refresh.

---

## Built With
*HTML
*CSS
*Javascript
*JQuery
*Moment.js
*Bootstrap
